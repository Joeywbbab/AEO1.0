## 互联跳转总览

- **Dashboard → Chats 详情**：`Recent Chats` 表格行点击跳转到 `analytics/chats/{chatId}`。
- **Dashboard → Guidelines**：右上角按钮固定指向 `/guidelines`。
- **Prompts → Prompt 详情**：表格行（非多选模式）点击跳转 `analytics/prompts/{promptId}`。
- **Prompts → Guidelines**：顶部按钮指向 `/guidelines`。
- **Citations (Domain 视图) → 域名详情**：表格行点击跳转 `analytics/citations/domain/{domain}`。
- **Citations → 外部站点**：行内外链按钮在新标签打开 `https://{domain}`。
- **Citations → Guidelines**：顶部按钮指向 `/guidelines`。
- **Competitors → Prompt 详情**：`Visibility Contributors` 展开后的 Prompt 行点击跳转 `analytics/prompts/{promptId}`。
- **Audit 页面**：当前无跨页跳转，仅本页操作。
